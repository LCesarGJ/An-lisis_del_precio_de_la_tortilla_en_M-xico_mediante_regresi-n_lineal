# An-lisis_del_precio_de_la_tortilla_en_M-xico_mediante_regresi-n_lineal
Este proyecto analiza la evolución del precio de la tortilla en México utilizando modelos de regresión lineal. El objetivo principal es evaluar la relación entre el precio por kilogramo y el paso del tiempo, así como estimar el precio promedio esperado para el siguiente año.
